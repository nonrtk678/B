ตั้งค่า

การอ่านสลิป — เลือกได้ระหว่างแบบออฟไลน์ (ฟรี) หรือใช้ API เพื่อความแม่นยำสูงขึ้น
ปล่อยว่างไว้ = ใช้การอ่านในเครื่องแบบออฟไลน์ ฟรี ไม่ส่งข้อมูลออกไปไหน (เหมาะกับสลิปที่ตัวเลขคมชัด)
ใส่คีย์ = อ่านแม่นยำขึ้นด้วย AI แต่มีค่าใช้จ่ายตามอัตราของ Anthropic · คีย์เก็บในเครื่องนี้เท่านั้น

ข้อมูลและการสำรอง

ข้อมูลเก็บอยู่ในเครื่องนี้ ควรส่งออกเก็บไว้เป็นระยะเพื่อกันข้อมูลสูญหาย
JSON = สำรองครบทุกอย่างรวมไฟล์แนบ (ใช้กู้คืน/ย้ายเครื่อง) · CSV = เปิดใน Excel ได้ (ไม่รวมไฟล์แนบ)
